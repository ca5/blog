Published: 金 28 3月 2008
By admin

In web.

さくら鯖でダイナミックパブリシング全然うまくいかないのであきらめました

以下やったことメモ

1.ググったら散々出てくる、噂の.htaccessの編集
http://www.google.co.jp/search?q=sakura+movable+%E3%83%80%E3%82%A4%E3%83%8A%E3%83%9F%E3%83%83%E3%82%AF&hl=ja&lr=lang_ja&safe=off&client=firefox-a&rls=org.mozilla:ja-JP-mac:official&hs=Xjp&start=10&sa=N

2. *.php *.pm *.pl の全パーミッションを755 へ
mtを置いてるディレクトリ下で
"find ./ -name *.php -exec chmod 755 {} ¥;"
"find ./ -name *.pm -exec chmod 755 {} ¥;"
"find ./ -name *.pl -exec chmod 755 {} ¥;"
を実行

3. 再構築
